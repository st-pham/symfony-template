# Template pour la formation sensio
Symfony 6.0  
PHP 8.0  
NGINX  
MYSQL 5.7

### Prérequis
Docker, Docker Compose

### Démarrer le projet
```shell
bin/dev/start
```

To rebuild images
```shell
bin/dev/start --build
```
Le site sera disponible à http://localhost:8000/
### Composer
Par exemple :
```shell
bin/dev/composer --version
```

### Arrêter le projet
```shell
bin/dev/stop
```
